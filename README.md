# projetoNovoWebsiteInergiae
## Projeto de renovação do antigo website (feito com Wordpress) da empresa Inergiae.

A ideia deste repositório é tentar realizar uma completa reformulação do código usado para interface do website da Inergiae, empresa catarinense focada na fabricação de fontes de alta voltagem.

### Requisitos que o website deve cumprir:
- Trazer uma réplica da interface antiga para um código novo feito com HTML e React;
- Funcionar bem com o hosteamento do Hostgator, recebendo corretamente os emails enviados no contato;
- Criar uma opção de login e registro dentro do site para que o dono possa entrar em sua conta para poder editar o conteúdo das postagens;
- Criar uma forma interativa que replique o sistema de posts do Wordpress para que o dono do website possa criar postagens no blog próprio;
- Reformular o código de forma que ele se adapte bem a dispositivos móveis, como celulares.

### Outros objetivos possíveis:
- Tentar replicar os domínios extras (para versões em inglês e espanhol) com um script para tradução automática com o código do próprio site principal.
- Melhorar a segurança e responsividade do site, removendo o excesso de plugins do Wordpress, também realizando as funções dos mesmos com o código do próprio HTML.
